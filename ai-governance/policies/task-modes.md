# Task Modes Policy

This project supports three task modes that change context, scope defaults, and output structure.

## Diseño de sistema
- Default scopes: Architecture, Dependencies, Workflow, Security
- Output contract: ai-governance/policies/output-contracts/design-system.md
- Response must include: Refined Task, Assumptions, Architecture Overview, Modules, Entities / Data Model, Implementation Plan

## Debugging
- Default scopes: Troubleshooting, Observability, Workflow
- Output contract: ai-governance/policies/output-contracts/debugging.md
- Response must include: Refined Bug Statement, Assumptions, Possible Causes, Debug Plan, Likely Root Cause, Suggested Fix, Regression Checks

## Feature nueva
- Default scopes: Architecture, Workflow, Dependencies, Security
- Output contract: ai-governance/policies/output-contracts/new-feature.md
- Response must include: Refined Feature Request, Assumptions, Affected Modules, Design Impact, Implementation Plan, Files to Modify, Risks / Edge Cases
